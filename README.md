# Student Result Management System

## Description
A Python-based project developed for Class XII Computer Science practicals to manage and analyze student exam results.

## Features
- Password protected
- Add student records
- Make changes in records
- Remove any record
- Calculate total and percentage
- Display results
- Generate result in PDF format
- Predict final exam marks using ML
- Simple menu-driven interface

## Technologies Used
- Python 3

## How to Run
1. Clone the repository
2. Run `main.py`
3. Follow on-screen instructions
